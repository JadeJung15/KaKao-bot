## 짧은 설명

픽셀곰 운영방 알림을 봇 서버와 로컬 JS 자동응답으로 연결하는 전용 브릿지 앱

## 전체 설명 초안

픽셀곰 브릿지는 등록된 카카오 오픈채팅 운영방 알림을 픽셀곰 운영봇 서버와 로컬 JS 자동응답으로 연결하는 관리자용 브릿지 앱입니다.

앱은 사용자가 직접 허용한 알림 접근 권한을 통해 카카오톡 알림의 방 이름, 발신자명, 메시지 내용을 읽고, 등록된 운영방으로 확인된 알림만 서버로 전송합니다. 서버에서 응답이 반환되면 카카오 알림의 답장 기능을 통해 운영봇 응답을 보낼 수 있습니다.

서버 응답이 없는 경우에는 사용자가 앱 안에 저장한 JS 자동응답 스크립트가 보조 응답을 만들 수 있습니다. 이 앱은 일반 채팅 수집 앱이 아니며, 설정된 픽셀곰 운영방 외 알림은 전송하지 않습니다.

## 심사 메모 초안

이 앱은 카카오톡 오픈채팅 운영방의 관리자용 봇 브릿지입니다. 핵심 기능은 등록된 운영방 알림을 서버와 로컬 JS 자동응답으로 전달하고 생성된 응답을 알림 답장 액션으로 보내는 것입니다. 이를 위해 Android Notification Listener 권한이 필요합니다. 앱은 접근성 권한과 화면 감지를 사용하지 않으며, 등록된 방 이름과 일치하지 않는 알림은 서버로 전송하지 않습니다.

테스트 순서:
1. 앱 실행
2. 알림 접근 권한 허용
3. 구매자 가이드의 앱 연결코드로 자동 설정 실행
4. 앱의 서버 테스트 전송 실행
5. 앱의 JS 테스트 실행
6. 등록된 카카오 운영방에서 `/상태`, `/브릿지`, `/js상태` 입력
7. 서버 또는 JS 응답이 카카오 알림 답장으로 전송되는지 확인

## Data Safety 초안

수집/전송 항목:

- 앱 활동: 앱 전송 로그
- 메시지: 등록된 운영방의 카카오 알림 메시지 내용
- 개인 정보: 카카오 알림에 표시되는 닉네임

사용 목적:

- 앱 기능 제공
- 운영방 봇 자동 응답
- 장애 진단
- 사용자가 직접 저장한 로컬 JS 자동응답 실행

공유:

- 픽셀곰 운영봇 서버로 전송
- 제3자 광고/마케팅 공유 없음

보안:

- HTTPS 전송
- 등록된 방 이름과 roomId 기준 필터링
