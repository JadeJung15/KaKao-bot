API 연결 카카오 스킬과 브릿지 웹훅을 함께 받을 수 있습니다.

Features

방 운영에 바로 쓰는 픽셀곰 기능

운영자가 일일이 기억해야 했던 멤버 정보와 이벤트를 자동으로 모아, 필요한 순간에 명령어로 꺼내 볼 수 있게 구성했습니다.

프로필과 별명 관리

공식질문 양식, 프로필 등록, 별명 매칭으로 멤버 정보를 빠르게 찾습니다.

입퇴장 히스토리

신규 입장, 재입장, 퇴장, 강퇴 이력을 누적해 운영 판단 근거를 남깁니다.

닉네임 변경 추적

이전 닉네임과 현재 닉네임을 이어 보고, 고유값 기반 복구까지 준비합니다.

포인트와 순위

채팅, 출석, 좋아요, 이체, 레벨, 순위를 운영형 포인트 규칙으로 제공합니다.

관리자 권한

링크, 프로필, 포인트 지급과 차감처럼 민감한 작업은 관리자 명령어로 분리합니다.

원본 이벤트 로그

브릿지가 전달한 원본 값을 확인해 profileHash 같은 연결 단서를 검증합니다.

Daily Flow

입장 알림부터 운영 기록까지 한 흐름으로

  1. 이벤트 수신 입장, 퇴장, 강퇴, 닉네임 변경 이벤트를 브릿지에서 받습니다.
  2. 멤버 기록 갱신 프로필, 이름 이력, 포인트, 메시지함을 같은 방 기준으로 정리합니다.
  3. 명령어 응답 운영자는 카카오톡 안에서 필요한 조회와 관리를 바로 실행합니다.

기본 조회와 운영 관리, 포인트 기능을 역할별로 나눠 현장 사용성을 높였습니다.
