Setup

도입은 네 단계로 끝냅니다

  1. 브릿지 앱 설치 봇폰에 픽셀곰 브릿지를 설치하고 알림 접근 권한을 켭니다.
  2. 채팅방 등록 방 이름, 오픈채팅 링크, 입장확인 문구, 관리자 닉네임을 등록합니다.
  3. 방장봇 문구 설정 방장봇 환영메시지를 방별 입장확인 문구로 맞춥니다.
  4. 명령어 테스트 /브릿지, /상태, /출석, /포인트로 알림과 서버 연결을 확인합니다.
