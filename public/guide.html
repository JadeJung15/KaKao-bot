픽셀곰 공개 가이드

구매 전 확인 가능한 전체 흐름입니다. 상세 설치값과 라이선스 안내는 구매자 전용 가이드에서만 제공합니다.

1. 신청

회원가입 후 서비스 신청 화면에서 방 이름, 오픈채팅 링크, 관리자 닉네임을 입력해 신청합니다.

2. 승인

운영자가 입금 확인 후 방, 라이선스, 30일 구독을 승인합니다.

3. 구매자 전용 안내

승인된 계정은 구매자 전용 가이드에서 내부 테스트 앱 링크, 연결코드, 설치와 테스트 순서를 확인합니다.

4. 운영 시작

설정이 완료되면 카카오방에서 상태 확인, 출석, 포인트, 커스텀 명령어를 테스트합니다.
