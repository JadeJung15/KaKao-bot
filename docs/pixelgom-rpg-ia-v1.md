# 픽셀곰 RPG 모바일 앱 IA v1

작성일: 2026-05-28

## 1. 결론

픽셀곰 RPG는 세로형 모바일 자동전투 RPG로 설계합니다.
전투는 자동으로 진행하고, 사용자는 장비, 스킬, 펫, 편성, 성장 선택으로 개입합니다.
방치 보상은 재접속 동기를 만들고, 수집형 요소는 장기 성장과 소환/이벤트의 기반으로 둡니다.

| 항목 | 결정 |
| --- | --- |
| 화면 방향 | 세로형 모바일 |
| 장르 중심 | 자동전투 RPG |
| 보조 축 | 방치 보상, 수집 성장 |
| 하단 탭 | 마을, 모험, 캐릭터, 소환, 길드 |
| 카톡 연동 | 모바일 게임 전용 팩으로 별도 개발 |

## 2. 장르 차이와 적용 방식

| 구분 | 의미 | 픽셀곰 적용 |
| --- | --- | --- |
| 방치형 RPG | 앱을 꺼도 시간에 따라 보상이 쌓이는 구조 | 오프라인 골드, 경험치, 장비 재료, 타이포게임 보너스 수령 |
| 수집형 RPG | 캐릭터, 펫, 장비, 몬스터 도감 등을 모으는 구조 | 픽곰 직업, 펫, 장비 세트, 몬스터 도감, 이벤트 스킨 |
| 자동전투 RPG | 전투는 자동 진행, 사용자는 성장/편성/스킬로 개입 | 기본 전투 방식. 스킬 버튼과 자동 진행을 함께 제공 |

추천 구조는 `자동전투 RPG + 방치 보상 + 수집 성장`입니다.
세로형 화면에서 전투 조작을 복잡하게 만들면 피로도가 높아지므로, 전투는 자동 중심으로 두고 성장 선택을 재미의 핵심으로 둡니다.

## 3. 전역 화면 규칙

| 영역 | 기준 |
| --- | --- |
| 상단 HUD | 프로필, 레벨, 경험치, 골드, 보석, 알림/메뉴 |
| 중앙 영역 | 현재 탭의 핵심 플레이 화면 |
| 우측/좌측 퀵 버튼 | 이벤트, 출석, 우편함, 퀘스트, 상점, 설정 |
| 하단 탭 | 5개 고정 탭. 전투 중에는 스킬/보상 UI로 임시 전환 가능 |
| 팝업 | 보상 수령, 강화 결과, 소환 결과, 카톡 연동 안내 |

## 4. 하단 탭 구조

| 탭 | 핵심 역할 | 포함 화면 |
| --- | --- | --- |
| 마을 | 첫 화면과 일일 루틴 허브 | 출석, 우편함, 퀘스트, 방치 보상, 이벤트, 상점 바로가기 |
| 모험 | 전투와 스테이지 진행 | 월드맵, 챕터, 스테이지, 전투, 보스, 전리품 |
| 캐릭터 | 성장과 편성 | 캐릭터, 장비, 강화, 스킬, 펫, 전투력 |
| 소환 | 수집과 수익화 | 영웅/펫/장비 소환, 확률 안내, 소환 이벤트 |
| 길드 | 카톡 연동과 커뮤니티 | 길드 출석, 길드 보스, 타이포게임 랭킹, 길드 상점 |

## 6. 핵심 사용자 루프

```text
접속
-> 방치 보상 수령
-> 출석/우편/퀘스트 확인
-> 장비/스킬/펫 성장
-> 모험 스테이지 자동전투
-> 전투 결과 보상 획득
-> 강화/소환/도감/길드로 장기 성장
-> 카톡 타이포게임 또는 길드 활동으로 보조 보상 획득
```

## 7. 카톡 연동 위치

카톡 연동은 앱의 중심 플레이를 대체하지 않습니다.
앱에 접속할 이유를 늘리는 보조 루프로 둡니다.

| 연동 지점 | 앱 화면 | 카톡 기능 |
| --- | --- | --- |
| 계정 연결 | 설정, 길드 | `/픽곰연동` |
| 일일 보너스 | 마을, 길드 | `/오늘보상`, `/길드출석` |
| 타이포게임 | 길드, 이벤트 | `/타이포게임`, `/타이포랭킹` |
| 쿠폰 | 우편함, 이벤트 | `/쿠폰` |
| 랭킹 공유 | 길드, 랭킹 | 방/길드 순위 요약 |
