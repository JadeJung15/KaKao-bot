# AGENTS.md

## 응답 규칙

- 한국어 존댓말로 답변합니다.
- 결론과 실행안부터 제시하고, 불필요한 배경 설명은 줄입니다.
- 작업 결과는 실무자가 바로 확인할 수 있게 표, 체크리스트, 단계별 요약으로 정리합니다.

## 프로젝트 기준

- 이 프로젝트는 repo 이름에 sunofox가 포함되어 있어도 WEBLING 제작부 통합 분석 시스템 개발로 취급합니다.
- 회사 BO 데이터를 제작부 실무용으로 가공, 확인하는 분석 사이트이며 production, defect, return-defect, BO 브릿지, Firebase, Cloudflare 배포가 주요 범위입니다.
- 현재 Codex는 Full Access 기준으로 사용하지만, 요청 범위 밖 작업은 하지 않습니다.
- 대규모 리팩토링, 임의 구조 변경, 불필요한 파일 수정은 금지합니다.

## 작업 시작 전 확인

- 현재 브랜치, git status, 최근 커밋, 원격 저장소를 먼저 확인합니다.
- 수정 대상 파일과 배포 상태를 먼저 확인합니다.
- 코드 수정 전에는 수정 예정 파일과 영향 범위를 요약합니다.
- 명확한 소규모 작업은 요약 후 바로 진행할 수 있습니다.
- 위험 작업이나 메인 반영 작업은 사용자의 별도 지시를 기다립니다.

## 배포 및 권한 제한

- 기본 배포는 Firebase dev 서버까지만 허용합니다.
- Cloudflare 메인 배포, Firebase Functions 배포, Firestore rules 변경, 인증정보 저장, 강제 push, 무단 커밋은 별도 명시 없이는 금지합니다.
- 테스트 통과 전에는 다음 단계로 진행하지 않습니다.

## Play Console 작업 규칙

- Play Console 업로드는 비공개 테스트만 사용합니다.
- Play Console 로그인, AAB 업로드, 변경사항 게시는 Codex 자동화 테스트 Chrome, Playwright, Chrome DevTools, browser-use로 진행하지 않습니다.
- Google 로그인은 자동화 제어 Chrome을 안전하지 않은 브라우저로 감지하여 차단할 수 있습니다.
- Play Console 작업의 1순위 방식은 이미 로그인된 일반 Chrome 창을 Codex Chrome Extension(Control Chrome with Codex)으로 직접 조작하는 방식입니다.
- Codex가 브라우저 조작을 보조해야 하는 경우에는 자동화 테스트 브라우저, Chrome DevTools/CDP, Playwright 브라우저, browser-use 대신 Codex Chrome Extension으로 기존 Chrome 탭을 잡아 진행합니다.
- Codex Chrome Extension 연결이 되지 않으면 다른 브라우저 자동화로 우회하지 말고 사용자에게 중단 사유와 필요한 조치를 먼저 보고합니다.
- Codex는 업로드 전 검증, AAB 경로, SHA256, 버전 확인, 업로드 후 체크리스트 안내를 수행할 수 있습니다.
- 반복 자동화가 필요하면 Play Developer API와 서비스 계정 방식으로 별도 승인 후 진행합니다.

## UI/UX 작업 용어 기준

| 표현 | 의미 |
| --- | --- |
| UI 개선 | 화면 디자인, 버튼, 색상, 레이아웃 개선 |
| UX 개선 | 사용 흐름, 이해도, 편의성 개선 |
| UI/UX 리디자인 | 화면과 사용성을 같이 다시 설계 |
| 사용성 개선 | 사용자가 헷갈리지 않게 만드는 작업 |
| 정보 구조 개선 | 메뉴, 설정, 기능 배치를 정리하는 작업 |
| 디자인 시스템 정비 | 버튼, 카드, 색상, 폰트, 간격을 통일 |
| 온보딩 개선 | 처음 설치한 사용자가 쉽게 시작하게 만드는 작업 |
| 접근성 개선 | 글자 크기, 대비, 터치 영역, 가독성 개선 |
| 인터랙션 개선 | 누르면 어떻게 반응하는지, 상태 표시 개선 |
| 마이크로카피 개선 | 버튼명, 안내문, 오류 메시지 문구 개선 |

- 구매자 화면은 온보딩, 접근성, 마이크로카피를 우선합니다.
- 관리자 화면은 정보 구조, 사용성, 인터랙션 피드백을 우선합니다.
- 상세 기준은 `docs/ui-ux-work-types.md`를 따릅니다.
